# Bean — мастер-план разработки веб-приложения

> **Назначение:** единый навигационный документ по всей работе. После одобрения копируется в `frontend/docs/plan.md` и обновляется после каждого этапа (чекбоксы, лог решений). Работаем строго по этапам: **вопросы по странице → мок-данные по схеме → реализация → проверка → коммит**.

## Контекст

Строим **Bean** — русскоязычную платформу, где авторы ведут платные сообщества (курсы, посты, чаты) и зарабатывают на подписках. Кодовая база — Next.js 15 (App Router) + Supabase + React Query + Tailwind v4, архитектура FSD (`app → pages → widgets → features → entities → shared`).

**Источники истины:**
- Дизайн-система: **сам код** (`app/globals.css` = токены, `src/shared/components` = кит) + карта [DESIGN.md](../DESIGN.md). Папка `bean-design-system/` **удалена** на этапе 17 — на неё больше не ссылаться
- UX-референсы: Discord (структура сообщества, чат) и Whop (монетизация, витрина)
- Контракт данных: `frontend/docs/db-schema.md` (утверждён; моки повторяют его форму)
- Копирайтинг: русский, sentence case, обращение на «вы», без эмодзи

## Глобальные правила (утверждены пользователем)

| Решение | Значение |
|---|---|
| Бренд | **Bean** (лаймовый боб), ProFound удалён |
| Данные | **Сначала UI на моках** по форме схемы; реальные миграции + подключение БД — предпоследний этап |
| Платежи | **Симуляция**: «оплата» мгновенна, транзакции пишутся в БД; реальный провайдер — после v1 |
| Тарифы | Полная гибкость: recurring с любым периодом/ценой/скидкой, one-time платёж за вход, бесплатно = без активных тарифов |
| Комментарии | Раскрывающаяся лента под постом, один уровень |
| Чат v1 | Отправка + редактирование/удаление своих + модерация админом. Реакции/reply — v2 |
| Видео курсов | Supabase Storage (bucket `lesson-videos`), HTML5-плеер, signed URLs |
| OAuth VK/Яндекс | Скрыт до реальной интеграции |
| OTP | 6 цифр (дефолт Supabase), стилизация по DS |
| Редирект после входа | Последнее сообщество (`localStorage bean:last-community`) → первое → пустое состояние с CTA |
| Rail-кнопки «сообщения»/«поиск» | Скрыты до v2 |
| Аватары без лого | Инициалы на детерминированном пастельном фоне (Discord-style), общий компонент `Avatar` |
| Модалка создания | Без загрузки лого (лого — в админке «Внешний вид») |
| Роли | Модель A: инлайн-переключатель «Просмотр ↔ Редактирование» для админа |
| Адаптив | Desktop-first без жёстких ширин; мобильная версия — этап 14 |
| Git | Коммит после каждого завершённого и проверенного этапа (`feat: этап N — …`) |

## ✅ Этап 7 — Курсы (готово)

**Решения:** последовательное прохождение (урок открыт после отметки предыдущего пройденным); видео опционально (текстовые уроки поддерживаются); в мок-режиме все видео играют общий демо-ролик (BigBuckBunny), а «загрузка» пишет фиктивный `video_path` + длительность из метаданных файла; новый таб-курс создаётся пустым (без сида).

- Entity [course](src/entities/course/index.ts): mock-store форм `courses`/`course_modules`/`course_lessons`/`lesson_progress`; курс создаётся 1:1 с табом при первом открытии; маппер считает сквозную нумерацию, прогресс и последовательную разблокировку
- **Плеер** ([LessonView](src/pages/community-channel/ui/course/LessonView.tsx)): видео/текстовый блок, пиллы «Урок N из M» + длительность, Назад / Следующий урок (недоступен, пока текущий не пройден) / «Отметить пройденным» (toggle)
- **Список уроков** ([LessonList](src/pages/community-channel/ui/course/LessonList.tsx)): прогресс-бар N/M, модули «Модуль N · название», строки с check/play/lock и таймингом Geist Mono; заблокированные некликабельны
- **Конструктор (модель A)** — [SegmentedControl](src/shared/components/SegmentedControl.tsx) «Просмотр ↔ Редактирование» (новый компонент UI-кита по DS): инлайн-переименование/удаление модулей, «+ Добавить модуль/урок» (урок создаётся сразу и открывается в редакторе), [LessonEditor](src/pages/community-channel/ui/course/LessonEditor.tsx) — название/описание (RHF+zod), загрузка видео с чтением длительности (`getVideoFileDuration`), удаление с конфермом
- Утилиты: `formatDuration` (мм:сс), `getVideoFileDuration`

**Осталось вне v1:** перестановка модулей/уроков (DnD), материалы-файлы к уроку, реальная загрузка в Storage.

## Этап 17.6 — Навигация и загрузка: мгновенные переходы, скелетоны, префетч, сеть, SSR витрины
