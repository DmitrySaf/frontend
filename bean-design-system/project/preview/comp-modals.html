Тип 2 · Подтверждение действия — нейтральное · без риска данных

Поведение · движение · доступность
primary — основное действие destructive — необратимое warning — потеря несохранённого success — результат lime — брендовые вехи
Иерархия действий. Одно primary на модалку, справа (end) или внизу (stack). Деструктивное всегда отделено от безопасного — split 50/50, либо разнесено по краям (spread). Никогда не ставьте две solid-кнопки рядом.
Движение. Появление: scrim fade 120ms; карта — opacity + translateY 8px → 0 + scale .98 → 1, 180ms, ease-out. Bottom sheet — slide снизу 220ms. Закрытие — обратно, 140ms. При prefers-reduced-motion остаётся только fade.
Закрытие. Esc и клик по scrim закрывают нейтральные диалоги; destructive и формы с вводом — только явными кнопками (защита от случайной потери). Full-screen — кнопкой Закрыть.
Доступность. role="dialog" + aria-modal, focus trap внутри карты, фокус возвращается на триггер. Первый фокус — на основное поле (формы) или на безопасную кнопку (confirm). Контраст текста ≥ 4.5:1, hit-target кнопок ≥ 44px.
