# Технический контекст проекта ProFound

## Архитектура

- **Frontend**: Next.js с App Router
- **Стилизация**: Tailwind CSS
- **Типизация**: TypeScript
- **Компонентная библиотека**: собственные UI-компоненты

## Структура проекта

- `/app` - страницы и маршруты
- `/components` - переиспользуемые компоненты
- `/components/ui` - базовые UI-компоненты
- `/lib` - утилиты, хуки, типы
- `/public` - статические ресурсы

## Ключевые технические решения

- **Серверные и клиентские компоненты**: оптимизация производительности
- **Адаптивный дизайн**: Mobile First подход
- **Типизация данных**: строгая типизация с TypeScript
- **Оптимизация изображений**: использование Next.js Image
- **Доступность**: семантическая верстка и ARIA-атрибуты

## Инструменты разработки

- **Пакетный менеджер**: pnpm
- **Форматирование кода**: Biome
- **Управление версиями**: Git

## Планируемые технические задачи

- Выбор и интеграция системы авторизации
- Разработка бэкенда (вероятно, с использованием API Routes Next.js)
- Выбор системы хранения и управления контентом
- Реализация системы платежей и подписок

## Технические требования

- Быстрая загрузка страниц
- Оптимизация для SEO
- Поддержка всех современных браузеров
- Масштабируемость для большого количества пользователей и контента
