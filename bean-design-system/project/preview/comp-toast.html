Toast vs Alert: alert встроен в страницу и постоянен; toast приподнят, временный и сам исчезает. Появляется по короткому ease-out, авто-скрытие через ~4–6 c (полоса-таймер), действие «Отменить» продлевает показ. Одиночная строка центрируется с иконкой; при описании или кнопках контент выравнивается по верху.
