# Руководство по стилю проекта ProFound

## Общие принципы

- Использовать русский язык для UI элементов и текстов
- Использовать английский язык для кода, переменных и комментариев
- Стремиться к простоте и понятности интерфейса
- Обеспечивать доступность для всех пользователей
- Следовать принципам Mobile First

## Именование

- Компоненты: PascalCase (например, `AuthorProfile`)
- Функции и переменные: camelCase (например, `getUserData`)
- Константы: UPPER_SNAKE_CASE (например, `MAX_FILE_SIZE`)
- Файлы компонентов: PascalCase (например, `AuthorCard.tsx`)
- Утилиты и хуки: camelCase (например, `useSubscription.ts`)

## Структура проекта

- `/app` - страницы и маршруты
- `/components` - переиспользуемые компоненты
- `/components/ui` - базовые UI-компоненты
- `/lib` - утилиты, хуки, типы
- `/lib/types` - типы TypeScript
- `/lib/constants` - константы
- `/public` - статические ресурсы

## Компоненты

- Разделять компоненты на презентационные и контейнерные
- Использовать серверные компоненты где возможно
- Клиентские компоненты помечать директивой `'use client'`
- Избегать больших компонентов, разбивать их на меньшие
- Для форм использовать React Hook Form

## Стилизация

- Использовать Tailwind CSS для стилизации
- Избегать инлайн-стилей
- Для сложных компонентов использовать композицию классов с помощью `clsx` или `cn`
- Придерживаться дизайн-системы (цвета, отступы, типографика)

## Цветовая палитра

- Основной цвет: `#3B82F6` (синий)
- Акцентный цвет: `#10B981` (зеленый)
- Цвет предупреждения: `#F59E0B` (оранжевый)
- Цвет ошибки: `#EF4444` (красный)
- Цвет текста: `#1F2937` (темно-серый)
- Цвет фона: `#FFFFFF` (белый)
- Цвет второстепенного текста: `#6B7280` (серый)

## Типографика

- Основной шрифт: Inter
- Заголовки:
  - H1: 2.25rem (36px)
  - H2: 1.875rem (30px)
  - H3: 1.5rem (24px)
  - H4: 1.25rem (20px)
- Основной текст: 1rem (16px)
- Второстепенный текст: 0.875rem (14px)

## Отступы и размеры

- Использовать систему размеров Tailwind
- Базовый отступ: 1rem (16px)
- Внутренние отступы компонентов: 1rem (16px)
- Отступы между компонентами: 1.5rem (24px)
- Скругление углов: 0.375rem (6px)

## Доступность

- Использовать семантические HTML теги
- Добавлять ARIA-атрибуты где необходимо
- Обеспечивать навигацию с клавиатуры
- Поддерживать высокий контраст для текста
- Текст должен быть читаемым (минимальный размер 14px)
