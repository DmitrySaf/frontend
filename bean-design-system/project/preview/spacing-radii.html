Corner radii

Единая лестница скруглений — от мелких элементов к крупным поверхностям. Базовое значение xl · 16px (кнопки, карточки, поля). Шаг 2px в рабочем диапазоне; full — для полностью круглых элементов.

Скругление должно расти вместе с размером элемента: чем крупнее поверхность, тем больше радиус. Внутри одного компонента не смешивайте соседние ступени — выберите одну. full используется только для пилюль, тогглов и аватаров, а не для прямоугольных карточек.
